## 目标概述
- 构建一个 React 前端驱动的视频剪辑架构，能根据「视频特征」与「用户剪辑诉求」自动生成剪辑方案并执行输出。

## 架构设计（高层）
### 1) 数据与特征层
- 视频特征提取：镜头分割、节奏/音量峰值、人脸/主体检测、关键帧、运动强度等。
- 诉求建模：剪辑类型、时长、风格、节奏、保留主体/片段优先级、音画同步策略。

### 2) 规则/策略层
- 策略引擎：将特征与诉求映射为剪辑策略（如保留高能镜头、优先含人脸片段）。
- 评分与排序：为候选片段打分、做时长约束和时间线拼接。

### 3) 生成与执行层
- 时间线生成：根据策略输出可执行的剪辑时间线（track/clip/effect）。
- 执行与渲染：通过前端渲染（预览）与后端/本地转码（导出）。

### 4) React 视图层
- 交互：需求输入、策略预览、时间线编辑与回退。
- 可视化：特征可视化、剪辑理由提示、实时预览。
